3.7. UpstreamConnectionService

Because the Foglight Agent Manager uses a polling model to connect to its upstream server, the upstream server can dynamically adjust the polling interval to account for the load currently being placed on it by all its downstream connections. As the load increases, all Foglight Agent Managers connecting to that server are instructed to decrease the frequency at which they connect and submit data. Conversely, when the load decreases, downstream connections are instucted to increase the frequency at which they connect and submit data.

Agents (data collectors) can use the UpstreamConnectionService to be notified of changes to the polling interval. These agents can adjust the amount of data they collect and the collection frequency to correspond to the load currently being placed on the upstream server.
