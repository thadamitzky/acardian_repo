3.1. Overview

When writing agents, there are a number of services provided by the FglAM host that can be used by an agent. Ideally, anything that does not involve collecting data can be done using the provided agent services.

Not all provided services are dicussed here. Please refer to the Javadoc provided with the developer's kit for the complete list.
