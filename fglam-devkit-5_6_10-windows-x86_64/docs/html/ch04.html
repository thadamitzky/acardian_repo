4.1. Overview

FglAM tooling has been growing since the first release (which was named Glue and ran as a SPID process). In this release, tooling has been updated to provide some new features, including Agent Simulators.

FglAM agent tooling allows agent developers to concentrate on building their monitoring components, and allows the code generators to create the Foglight configuration set. The tooling is optional, but we highly recommend that you use it in your agent development cycles.
