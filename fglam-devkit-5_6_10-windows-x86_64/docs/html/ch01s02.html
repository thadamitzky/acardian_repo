1.2. Skills Required by Foglight Custom Agent Developers

Building Foglight agents is a custom software development project. As such, skilled personnel are required to build and manage the code over the lifecycle of the project. The requirements for building and managing Foglight custom agents include, but are not limited to:

  • Expert knowledge of the domain for which you are building a new agent.
  • Expert knowledge of how to use a Java interface to retrieve the desired metrics from that domain.
  • The ability to code, debug and troubleshoot in Java.
  • The ability to use Apache Ant to automate the software build process.
  • An environment for unit and regression testing.
  • A commitment to regression testing the agent during Foglight upgrades.
