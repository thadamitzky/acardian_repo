5.2. API Commitments

The following are our API commitments:

  1. Published interfaces will be supported for as long as possible. Old interfaces will be deprecated as they are replaced with new ones, but will only be removed under critical circumstances. A removal warning will be made at least six months before a published interface is removed from a future version of FglAM.
  2. If new methods need to be added or an existing method changed, a new interface will be created.
  3. If a method name is the same for two different interfaces, then their behavior is the same. If the behavior of a method must be changed, a new interface with a new method name will be created.
