1.3. Build Platform Prerequisites

Before you begin the development process for building agents with the Foglight Agent Manager Developer's Kit, a number of prerequisites must be installed and configured.

  • Java JDK: Agents developed with the Developer's Kit are built primarily with Java. The Foglight Agent Manager runs under a 1.6 JRE. Also, it is required that you build and compile your agent with the 1.6 JDK. This JDK can be downloaded here.
  • Apache Ant: The latest version of Apache Ant (which is 1.7.1, at the time of writing) can be downloaded here. Once set up and configured, Ant is used to build all Developer's Kit artifacts.
  • OS Platform: Since agents developed with the Developer's Kit are Java-based, you may build and design your agent on any operating system platform that supports the required Java JDK version.
