6.1. Overview

FglAM 5.5.0 introduced two new features for collecting data and controlling agents: structured data collectors and agent callbacks.

To support structured data collection and callbacks, two interfaces were added to the FglAM agent API: CollectorAware and CallbackAware.

Tooling automatically creates extensions of these two interfaces. FglAM will inspect your agent for any interfaces that extend these two and then set up your agent appropriately.
