1.1. Overview

The Foglight Agent Manager manages agents installed on monitored hosts, providing a centralized communications link between the Foglight Management Server (FMS) and the agents. The Foglight Agent Manager also provides a number of support services such as installation, upgrade, and the ability to configure agents.

This guide provides documentation for the Foglight Agent Manager Developer's Kit (devkit). In addition to this guide, the following resources are accessible from the <devkit install directory>/docs/index.html file:

  • ChangeLog for the Developer's Kit: The change log for the devkit on a release-by-release basis.
  • Developer's Kit API Javadoc: The complete reference guide for the public java interfaces provided by the FglAM API.
  • Agent Definition XSD Docs: The complete reference guide for the agent definition XML file format.
  • Agent Manifest XSD Docs: The complete reference guide for the agent manifest XML file format.
  • Message XSD Docs: The complete reference guide for the XML format used by the FglAM communication layer.
