4.3. Agent Definition

The Agent Definition XML is the core source document for use by the Foglight Agent Manager tooling code generators. The Agent Definition contains all of the agent-specific configurations for each of the agent definitions within your package.

You can explore further details on the Agent Definition by reviewing the XSD .
