The list of topology defintion files that will be added to the core (server) topology types.
If any topology type name matches a server topology type, the original type will be
comletely replaced with the new type. If you use this, you will need to add a cartridge
lifecycle script to modify the topology definitions on the server. This file will not
be included in your cartridge and is only used for agent code generation.
</xsd:documentation>
